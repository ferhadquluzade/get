# get
An shell (terminal) utility to get mostly used data easily

<hr>

# Installing

```shell
git clone https://github.com/ferhadquluzade/get.git
cd get
npm install 
npm install -g 
```

# Usage
Usage: get --[FLAG]


```shell
# Ip address (public v4, local)
get --ip  
```
```shell
# CPU information
get --cpu 
```
```shell
# OS information
get --os  
```
```shell
# Memory/RAM usage 
get --ram  
```
```shell
# Environment variables 
get --env  
```
